# ForgeHub

ForgeHub is the **central console** for planning, governing, and executing software projects with AI agents. It replaces the previous “Hermes Agent Forge” naming throughout the codebase, UI and documentation.

- Register products, versions, and projects.
- Define pipelines, stages, and required artifacts.
- Break planning items into tasks and assign them to agents.
- Track execution, approvals, and audit events.
- Provide a clean, modern UI built with React‑Vite, Tailwind and shadcn/ui.

## Quick start
```bash
# Build and run the stack
docker compose up -d --build

# API health check
curl http://localhost:8000/health
# → {"status":"ok"}

# Frontend UI
open http://localhost:4173
```
